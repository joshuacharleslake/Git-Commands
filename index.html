Git Commands

Initialising git to a project

Create new project and on the root of your project in terminal type the following to initialise git

git init

This will install git to the project create a hidden file .git within your project.

Using Git

After creating your project files, open terminal and change directory into your project folder where you .git folder is stored.

Type the following command to see the status of your project.

git status

You should receive the following showing what files have been edited.

On branch master
Changes not staged for commit:
(use "git add file..." to update what will be committed)
(use "git checkout -- file..." to discard changes in working directory)

modified: css/styles.css
modified: index.html
no changes added to commit (use "git add" and/or "git commit -a")

To add the files separately type git add followed by what files you want to add.

git add css/styles.css
git add index.php

If you would like to add all the modified files, type the following:

git add .

Now if you was to enter the git status command again, you would see which files have been added in green.

On branch master
Changes to be committed:
(use "git reset HEAD file..." to unstage)

modified: css/styles.css
modified: index.html

To commit the changes to the repository type the following entering your commit message.

git commit -m "Commit Message"

Useful Commands

git log
git log --oneline

Above command shows a log of all your commits. 2nd like will display it on one line.

type q to quit

git diff

Above command shows the differences in the modified files.

git commit -a -m "Commit"

Above command adds and commits at same time.
